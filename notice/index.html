欢迎各位新同学！

只有 Windows 才有这种东西——Linux 从来不会蓝屏死机！因为崩溃叫做 Kernel Panic。

加入社团

因为我们是一个开放的社团，QQ 群是一个开放的群，因此群成员不一定是社团成员，据说老大哥也在群里面看着我们。因此新人请主动找我们报名，否则，我们无从知道你是社团成员还是校外的间谍游客。

面试

面试是用来踢人的，所以请放心——我们从来不用脚投票。

不过，我们知道，很多人刚来石油大学，只有没有人认识你，所以，我们总得互相认识一下吧。

为了互相认识，近期我们将举行见面会。时间地点待定。

强制性

我们都是搞技术的，不喜欢强行要求和被要求做事情，因此我们当然不会随便让你去跑龙套和领便当。

只不过，如果你长时间不活跃，线上和线下都不活跃，我们会认为你已经蒸发自行退出社团了。

活动通知

我们一直觉得“收到请回复”的短信是很令人讨厌的，因此我们会在 QQ 群发群公告。请在每个周五晚上之前看看有没有什么群公告。

如果不使用 QQ（例如使用电子邮件或者 Facebook），希望从短信或电子邮件等其他渠道获得消息，请一定让我们知道。 长时间与社团没有联系，我们会认为你已经主动退出了。

我什么都不懂

只要你去翻翻《算法导论》，你就知道我们研究的东西有多么浅显简单了。

我们每个人都是从零开始的，而且我们已经攻克了某些困难问题，因此实际上新人们的学习门槛更低了。

我不会编程

跟着我们，到时候你就知道应该怎样学编程了。没准你还会发现除了考二级，编程能够做一些实事儿比如强行装 13。

我的电脑

还没买？

我们当中就有内行——目前我们还没听说有哪个外行能够打赢与奸商的战争。

大一不让带电脑？

无论这条规定和早操早读晚自习是否不合理不合法，我们都应该知道这条命令的本意：不要沉迷电脑，影响学习。


一群正在跑早操的学生

不幸的是，Linux 和相关技术都是在一重启就全部消失的机房里不太容易实践的电脑技术，因此电脑是一个重要工具。

因此，我们的建议是：

带电脑事实上没有问题，不过在禁令解除之前，不要在宿舍用电脑，毕竟导员不会搜查你的柜子，也不会因为听某个探子汇报说你带了电脑就到图书馆或自习室抓捕教育你。

此外，我们还有三条忠告：

  1. 少在有着“时间杀手”美誉的宿舍呆着。
  2. 千万不要挂科。
  3. 也不要仅仅满足于考试及格。 如果每一科分数都不高，你照样毕不了业。

我们知道很多大一新生没有电脑因为我们大一时候也没有，因此我们可以稍稍转移一下话题——和 Linux 有关的内容太多了！

树莓派！

不知道树莓派是什么？这么讲吧，树莓派是风靡全球的极客的玩具，是玩起来像是和机器人协会做同样事情，实际上完全不一样的卡片电脑……


其实树莓派很好吃只要你牙口够硬（查看原文）

社团里有一些玩树莓派的，反正有些人的派也是闲着，为什么不借一个玩玩呢？土豪可以自己买，不过之前一定要向我们咨询一下，免得被奸商坑，或者弄到了一些不实用的东西。

反正导员只说“大一不让带电脑”，没说“大一不让带树莓派”，也没说“大一不许玩‘单片机’”（树莓派不是单片机）……

交流平台

QQ 群

  1. 请改一下群名片。如果坚持匿名，我们会怀疑您是有关部门请来的。
    据说这是有关部门为数不多的一次公开露面。
     
  2. 不要在群内发广告和宣传性文字，不要在群内发问卷等令人不愉快的东西。违者会被禁言。这一条是严肃的，这一条是严肃的，这一条是严肃的。（严肃的话说三遍）
  3. 重要的事情不要只在 QQ 群里面说，除非你想体验一下被遗忘的感觉。（重要的事情请打电话、发短信或发邮件。）

其他方式

我们已经清醒地认识到，光靠 QQ 并不能非常好地进行技术交流。

但是，由于国情特殊，目前我们还没有对除 QQ 以外的而且不会被墙的交流工具达成共识。

我们正在测试以下方案：

  • IRC（不能看记录）
  • 邮件列表（有一些问题）
  • BBS/论坛

由于这些方式均未达成共识，因此暂时我们仍然会把 QQ 当作线上“主战场”。

如果我们找到了一个比较好的交流工具，我们会在见面会的时候公布。

Google

遇到技术问题，遇到不知道的东西，在决定向人发难问之前，我们都会用 Google 搜一搜，虽然 Google 本身就是个技术问题。扯淡的时候才用百度。谷歌上不去的时候就拿必应代替，因为能搜英语……

Wiki

[% callout %] Wiki 不是维基百科

Wiki 不是维基百科（Wikipedia）。 [% endcallout %]

你知道吗，

  1. 很多 Linux 系统都有丰富的文档，例如 ArchWiki、Deepin Wiki。
  2. 中国人不喜欢写文档因为光是写公文就累得要死，所以很多时候英文资料更丰富。
  3. 各软件的黑历史可到维基百科中浏览。当然，英文的更有趣。

协会网站

我们特意把我们的黑历史知识经验做成了网站。

博客主要记录一些事件，当然也记录了一些人生的经验。不过，更多的内容都写在了入门指引 (https://upclinux.github.io/intro/)里——在我们共同努力之下，这些文字还有 3.1415926 个世纪就可以出版成书了。

群内交流

我们相信，如果我们都能够按照上面的提示，先行做一些准备工作，我们就可以更好地谈笑风生。

线下活动

不管你在群里是否潜水，线下活动当然要积极参加，刷存在感。

如果你长时间不活动，我们可能会把那个卖手抓饼的当作是你，你也有可能把那些打酱油的误认为是我们，互相之间不认识，这样就不好玩了。

GitHub

[% callout %] GitHub 是什么

问这个问题，说明你没仔细阅读前面内容——去 Google 查一下不就知道 GitHub 是什么东西了嘛！

为了给程序猿一点归属感，我们在这里恶搞了一个简介，当然，认真魔人也可以看，因为内容是可以切换的。 [% endcallout %]

社团网站全部都放在了 GitHub 里面 (https://github.com/upclinux)，而且还有一些其他的大小程序，全部开源。而且代码绝对不是火星人写的，因为人类能够理解。

本着自由开放的精神，我们欢迎大家恶搞参与编辑和贡献。
